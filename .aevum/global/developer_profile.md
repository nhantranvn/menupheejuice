# Developer Profile

This file describes the coding style, preferences, and communication habits of the developer.
The AI Agent should study this file to adapt its behavior and generate code that matches the developer's style.

## Coding Style
- Prefer concise, modern syntax (e.g., async/await over promises).
- Favor early returns over deeply nested if-statements.
- Keep functions small and focused on a single responsibility.
- Add minimal, meaningful comments to explain *why*, not *what*.

## Communication Style
- Keep responses brief and straight to the point.
- Show code rather than explaining it excessively.
- Do not repeat the prompt or explain basic concepts unless asked.

## Tooling & Workflow
- Standardize on npm for package management.
- Prefer explicit error handling at the boundary layers.
